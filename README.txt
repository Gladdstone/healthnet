---------------------------------------
INSTALLATION
---------------------------------------
1.) Download .zip file
2.) Extract to desired location
3.) Open terminal/cmd.exe to ~/healthnet
4.) Run command "python manage.py 
	runserver"
5.) Follow given URL/IP
---------------------------------------
---------------------------------------


---------------------------------------
BASIC EXECUTION
---------------------------------------
1.) If already registered, skip to step step 5.
2.) Select Register
3.) Fill out information
4.) Save
5.) Select Login
6.) Type in username and password, hit enter/return
7.) To schedule an appointment, select Create Appointment.
8.) To view appointments, select View Appointment.
9.) To view patient info, select View Profile.
10.) To logout, select Logout.
